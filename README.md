# Adventure Works Sales & Return Performance Dashboard

## Tools
- Power BI
- DAX
- Data Modeling

## Business Questions
1. How do sales and profit trend over time?
2. What are the best-selling products?
3. Which countries contribute the most sales?
4. Which product categories contribute the most revenue?
5. Which categories have the highest return rates?

## Key Insights
- Bikes contributed 57.27% of total sales.
- United States generated the highest revenue.
- Clothing had the highest return rate at 15%.
- Sales peaked in Q2 2023.

## Recommendations
- Focus marketing efforts on Bikes.
- Investigate high return rates in Clothing.
- Strengthen market presence in the United States.
